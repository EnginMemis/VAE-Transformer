# VAE-Transformer
GENERATING PARAPHRASED SENTENCES USING UNSUPERVISED TRAINING WITH VARIATIONAL AUTOENCODER MODELS

## MODEL TRAINING

```bash
python train_tokenizer.py \
       --data_name "corpus" \
       --data_dir "data" \ 
       --epochs 3 \
       --learning_rate 0.001 \
       --batch_size 64
```
